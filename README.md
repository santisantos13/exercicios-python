# 🐍 Exercícios Python

Repositório dedicado à prática diária dos meus estudos de Python,
enquanto curso Análise e Desenvolvimento de Sistemas (ADS) e Python 3 pela Udemy com o Prof. Otavio Miranda.

## 📂 Estrutura

O repositório é organizado por tema:

- `dicionarios/` — exercícios com dicionários e listas de dicionários
- `listas/` — manipulação e operações com listas
- `strings/` — manipulação de strings
- `funcoes/` — definição e uso de funções
- `poo/` — programação orientada a objetos
- `logica-condicional/` — estruturas condicionais e de repetição
- `banco-de-dados/` — estruturas de bancos de dados, SQLite/MySQL

## 🎯 Objetivo

Manter consistência na prática, documentar meu progresso e servir
como material de consulta pra mim mesmo no futuro.
